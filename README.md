# Portfolio Website

A modern, responsive portfolio website built using HTML, CSS, and JavaScript. This project showcases my skills, projects, and contact information while demonstrating responsive web design and interactive frontend features.

## Features

- Responsive design for desktop, tablet, and mobile
- Dark mode with preference saved using Local Storage
- Mobile navigation menu
- Smooth scrolling
- Scroll reveal animations
- Back to Top button
- Clean and modern UI

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
